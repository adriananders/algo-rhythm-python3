# Flask web server

1. Run server
2. Run clint build

Run Server

```sh
# Install
pip install Flask

# Start server (will be exposed)
sudo python server.py

# Open the site
open http://0.0.0.0:80/
```

Run client

```sh
npm i
npm run dev
```
